# Prompt: 美股期权卖方标的筛选

请帮我筛选适合卖出看跌期权（Sell Put）的美股标的，要求如下：

## 筛选条件

### 1. 趋势确认（必须全部满足）
- 当前股价 > 50日均线 > 200日均线（均线多头排列）
- 200日均线处于上升状态（至少连续上涨1个月）
- 股价距52周高点回撤不超过15%
- 近3个月相对标普500有正超额收益（相对强度为正）

### 2. 权利金要求（高隐含波动率）
- 30天期 ATM 期权隐含波动率(IV) > 30%（或 IV Rank > 50%）
- 月度期权链流动性好（bid-ask spread < 10%）
- 卖出1个月到期、Delta -0.25 至 -0.30 的 Put，年化收益率 > 15%
- 单手（100股）权利金 > $200

### 3. 安全性要求（排除暴雷风险）
- 股价在 $100 - $200 之间（单手保证金适中，权利金绝对值可观）
- 市值 > 100亿美元（大中盘，排除小盘投机股）
- 不是 meme 股或近期有被做空报告的公司
- 近4个季度 EPS 均为正（排除亏损企业）
- 资产负债率 < 70%（排除高杠杆）
- 不在财报公布前后5天内（避免 earnings event）
- 近1年最大回撤 < 20%（排除剧烈波动标的）
- 非生物医药/SPAC/中概股（排除二元事件风险）

### 4. 行业分散
- 最终推荐 5-10 只标的，行业尽量分散
- 注明每只股票所属行业（GICS分类）

## 输出格式

请按以下格式输出每只推荐标的：

| 字段 | 内容 |
|------|------|
| 代码 | ticker |
| 公司名 | 中英文 |
| 市值 | xx亿美元 |
| 当前价格 | $xx |
| 距52周高点 | -x% |
| 30天IV | xx% |
| IV Rank | xx% |
| 推荐卖出行权价 | $xx (Delta约-0.25) |
| 预估月度权利金 | $xx (年化收益 xx%) |
| 近1年最大回撤 | xx% |
| 上升趋势确认 | 50MA/200MA状态 |
| 下一财报日 | yyyy-mm-dd |
| 风险提示 | 简述该股最大风险因素 |

## 额外说明
- 我的风格是期权卖方（theta decay 策略），偏好"收租"而非方向性押注
- 我希望标的本身是我愿意持有的优质公司（万一被行权也能接受持股）
- 请用今天的实时数据，标注数据获取日期
- 如果当前市场环境整体波动率很低(VIX < 15)，请提醒我可能不是卖权的好时机
